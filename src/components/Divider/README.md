# Divider

Use Divider to horizontally separate similar content.

```vue
<template>
	<div>
		<h4>divider 👇</h4>
		<m-divider />
		<h4>divider ☝️</h4>
	</div>
</template>

<script>
import { MDivider } from '@square/maker/components/Divider';

export default {
	components: {
		MDivider,
	},
};
</script>
```
