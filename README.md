# Maker

Maker is a mobile-first themeable Vue.js 2.x component library. [View its docs here](https://square.github.io/maker/styleguide/latest-stable/#/).

## Install

```sh
# install maker
npm i @square/maker
# install peer dependencies
npx i-peers -a
```
